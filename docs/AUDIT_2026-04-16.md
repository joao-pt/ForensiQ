# Auditoria ForensiQ — 2026-04-16

**Escopo:** backend (Django 5 + DRF) + frontend (templates + vanilla JS) + segurança + performance.
**Veredito executivo:** base sólida com separação de camadas, imutabilidade tripla, auditoria com correlation_id, CSP, rate-limit JWT, hash-chain. Mas há vulnerabilidades reais (XSS via `unsafe-inline` + CSP, JWT em localStorage, IP spoofing em auditoria, `CORS_ALLOW_ALL_ORIGINS` acoplado a `DEBUG`) e problemas de correctness/arquitectura (hash-chain com nonce quebra verificabilidade, race em timestamps, N+1 latente, falta de índices em filtros, zero testes frontend).

---

## 1. Segurança

### 🔴 Críticos

- **S1 — JWT em localStorage + cookie não-HttpOnly** (`auth.js:32-33,80,170`). Qualquer XSS exfiltra sessão.
- **S2 — CSP com `'unsafe-inline'`** (`middleware.py:89-90`, `base.html:23`). Anula defesa XSS.
- **S3 — `CORS_ALLOW_ALL_ORIGINS = DEBUG`** (`settings.py:187`). Staging mal configurado = bypass.
- **S4 — `X-Forwarded-For` confiado sem whitelist** (`audit.py:28-30`). Cliente falsifica IP no AuditLog.
- **S5 — Hash-chain com nonce aleatório não gravado** (`models.py:502-504`). Hash irreproduzível = prova não verificável.
- **S6 — `integrity_hash` não cobre bytes da foto** (`models.py:250-264`). Substituição de imagem indetectável.

### 🟠 Altos

- **S7 — Sem CSRF em cookie JS-settable.**
- **S8 — Rate-limit anónimo global 10/min** (permissivo/restritivo).
- **S9 — Sem validação MIME real em uploads** (`models.py:29-41`).
- **S10 — `SECURE_PROXY_SSL_HEADER` sem guarda de proxy** (`settings.py:240`).
- **S11 — Admin URL só por obscuridade.**
- **S12 — Falta `Permissions-Policy`, `COOP`.**
- **S13 — JWT sem `SIGNING_KEY` separado de `SECRET_KEY`.**
- **S14 — Sem testes explícitos de IDOR/authz bypass.**

### 🟡 Médios

- **S15 — `ValidationError` do modelo vira 500** (`views.py:169-177`).
- **S16 — `UserDetailSerializer` não utilizado; PII exposto.**
- **S17 — EXPERT vê tudo** (`views.py:102-104, 150-155`).
- **S18 — CSP sem `object-src 'none'` nem `upgrade-insecure-requests`.**
